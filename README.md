# homebrew-tap

Personal Homebrew tap for Gunwoo Lee's tools.

## Install

```sh
brew tap leegunwoo98/tap
brew install leegunwoo98/tap/ghostty-tmux-attach
```

## Formulae

- [`ghostty-tmux-attach`](Formula/ghostty-tmux-attach.rb) — make Ghostty's `window-save-state` restore actually re-attach to tmux sessions instead of dropping users at a plain `$HOME` shell.
